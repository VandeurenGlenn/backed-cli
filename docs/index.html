    backed-cli

    A command line interface for fast es6 development

    Installation

    $ npm install --save --global backed-cli

    Usage

    • Create a 'backed.json' file in your projects root example

    Commands

    Build[--build, -b]

      backed --build

    Serve[--serve, -s]
